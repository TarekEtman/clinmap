# Evaluation Specification

This folder defines the public technical contract for the Clinical Model Behavior Evaluation Lab before the dataset is rebuilt.

Start here:

- [`clinical_model_behavior_eval_spec_v1.md`](clinical_model_behavior_eval_spec_v1.md)

The spec is intentionally stricter than the current MVP dataset. The next phase is to rebuild the data model around separated cases, responses, annotations, adjudications, source anchors, and run manifests.
