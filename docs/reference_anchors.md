# Public Reference Anchors

- OpenAI Evals cookbook: https://developers.openai.com/cookbook/examples/evaluation/getting_started_with_openai_evals
- Frontier Model Forum early best practices for frontier AI safety evaluations: https://www.frontiermodelforum.org/updates/early-best-practices-for-frontier-ai-safety-evaluations/
- Frontier Model Forum preliminary taxonomy of pre-deployment frontier AI safety evaluations: https://www.frontiermodelforum.org/updates/issue-brief-preliminary-taxonomy-of-pre-deployment-frontier-ai-safety-evaluations/
- Frontier Model Forum third-party assessments: https://www.frontiermodelforum.org/technical-reports/third-party-assessments/
