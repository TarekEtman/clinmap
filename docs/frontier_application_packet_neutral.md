# Frontier Evaluation Application Packet - Neutral Version

## One-line positioning

Licensed dentist and Global Health MPP building healthcare-domain model behavior evaluations: synthetic probes, response ranking, rubric calibration, failure taxonomy, validation scripts, charts, and reviewer-facing evidence.

## Technical proof link description

Clinical Model Behavior Evaluation Lab — **ClinMAP-VOI v0** (primary): synthetic metamorphic healthcare probes (40 families / 320 variants), hosted multi-model outputs, completed human domain review (3971 rows), metamorphic relation annotations, model metrics, and post-review QA audit with holdout families and explicit claim boundaries. Supporting lineage: 48-case synthetic v1 demo with explorer, validation harness, and PDF snapshot.

## Why this is relevant to frontier AI evaluation

Frontier model evaluation needs more than answer review. It needs people who can convert ambiguous risk into operational criteria: prompts, rubrics, provenance, scoring records, disagreement handling, metrics, and honest limitations. This project demonstrates that workflow in a healthcare-domain setting without using private data or overclaiming benchmark results.
