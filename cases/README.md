# Case Studies

These case studies use synthetic prompts and model-response patterns to demonstrate evaluation design, scoring, and failure analysis for healthcare-domain AI outputs.

| Case | Focus | Signal |
|---|---|---|
| [`01_false_reassurance_triage.md`](01_false_reassurance_triage.md) | High-risk symptom under-escalation | Clinical safety review and false reassurance detection |
| [`02_dental_medication_context.md`](02_dental_medication_context.md) | Oral health context + medication risk | Domain expertise, medication-context awareness, scope control |
| [`03_public_health_misinformation.md`](03_public_health_misinformation.md) | Misinformation + equity communication | Global health reasoning and evidence-quality review |
| [`04_reasoning_faithfulness_medical_qa.md`](04_reasoning_faithfulness_medical_qa.md) | Correct answer with unsupported rationale | Frontier-eval-adjacent reasoning faithfulness audit |

All cases are synthetic and are not clinical guidance.
